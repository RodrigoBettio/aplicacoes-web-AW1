Tag de Link

<a href="link"> texto do link </a>

Objetivo: Criar links para outras páginas ou seções da mesma página.

Atributos principais:

href: Endereço do link (URL).

rel: Define a relação entre o documento atual e o link (ex: "nofollow").

download: Força o download do arquivo no link.

name: Nomeia o link para ser usado como âncora.

type: Especifica o tipo de mídia do link.
